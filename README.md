# Arsenal-script

- **Aimbot:** Auto-locks on enemies.  
- **Circle Aimbot:** Locks instantly when any part enters the circle.  
- **ESP:** Enemies **red**, teammates **blue**.  
- **Team Mode:** No aiming at teammates.  

UI appears first, supports mouse interaction.
